CappRuby
--------
Ruby to run on cappuccino/objective-j written entirely in javascript (including
parser etc).

Installation
------------
Download/git clone and then:

    tusk install .

To run REPL:

    ./bin/cappruby

Requires narwhal, objj, cappuccino packages.
